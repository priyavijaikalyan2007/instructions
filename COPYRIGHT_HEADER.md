<!-- AGENT: Defines the required SPDX copyright header format for all source files. -->

Every source code file must contain SPDX compatible copyright headers as outlined on https://reuse.software/.

In particular ensure that:

- The date when the file was created is included in the copyright header.
- The header includes a unique GUID identifier for each file that is produced the first time it is generated.
- The header includes as many SPDX headers as needed, especially SPDX-FileCopyrightText and SPDX-License-Identifier.
- The license should conform to whatever license is included in the repository in the file LICENSE.md.
- Assume repositories with no LICENSE.md file imply proprietary source code.
- Include the repository name and the author name and email IDs in the copyright header.
- For proprietary software, include the company name which may be found in the file COPYRIGHT_INFO.md using the SPDX headers.
